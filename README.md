# Python mini projects 
This repository contains small Python projects built while learning programming fundamentals. 
The goal of this repository is to practice programming concepts and improve problem-solving skills.

## Structure
Each file focuses on a specific Python concept.

##  Projects

### 01 - Rock Paper Scissors 
A console-based Rock Paper Scissors game.

**Features:**
- Multiple rounds
- Score tracking
- Input validation
- Random computer moves

**Concepts practiced:**
- for loops
- conditional statements
- user input validation
- random module
